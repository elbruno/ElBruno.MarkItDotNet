# Sample Document

This is a sample EPUB document used for testing the MarkItDotNet library conversion capabilities.

## Features

* Convert EPUB to Markdown
* Support for **bold** and *italic* text
* Handle chapter structure

# Supported Formats

1. HTML (.html)
2. PDF (.pdf)
3. Word (.docx)
4. Excel (.xlsx)
5. PowerPoint (.pptx)
6. EPUB (.epub)

## Format Details

| Format | Extension | Supported |
| --- | --- | --- |
