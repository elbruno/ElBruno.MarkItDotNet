This is a sample plain text document used for testing the MarkItDotNet library.

Features:
- Convert multiple formats to Markdown
- Extensible converter architecture
- Support for satellite packages

Supported Formats:
1. HTML (.html)
2. PDF (.pdf)
3. Word (.docx)
4. Excel (.xlsx)
5. PowerPoint (.pptx)

For more information, visit: https://github.com/elbruno/markitdotnet
