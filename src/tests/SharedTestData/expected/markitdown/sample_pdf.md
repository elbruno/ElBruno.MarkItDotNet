Sample Document
This is a sample PDF document used for testing the MarkItDotNet library conversion capabilities.
Features
- Convert PDF to Markdown
- Support for text extraction
- Heading detection by font size
Supported Formats
1. HTML (.html)
2. PDF (.pdf)
3. Word (.docx)
4. Excel (.xlsx)
5. PowerPoint (.pptx)
Format Details
| Format | Extension | Supported |
| ------ | --------- | --------- |
| HTML   | .html     | Yes       |
| PDF    | .pdf      | Yes       |
| Word   | .docx     | Yes       |
